base58
======

This repository implements a very simple encode/decode conversion from base64 to base58 or vice versa.

### Build from source

Run the following instructions to build it.

```shell
$ mkdir build
$ cd build
$ cmake ..
$ make
```

### How to run

```shell
$ base58 --base64=BASE58_VAL
```
or

```shell
$ base58 --base58=BASE64_VAL
```
